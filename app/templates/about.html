About This Project

Multimodal Knowledge-Enhanced Prescriptive Maintenance with Explainable AI and Adaptive RAG

This is an advanced independent research and development project by Md. Ehsanul Haque Kanan. It demonstrates an end-to-end, intelligent system for Prescriptive Maintenance (PdM), evolving beyond traditional predictive models by integrating a sophisticated Retrieval-Augmented Generation (RAG) pipeline.

The system moves from simply predicting when an asset might fail to prescribing how to fix it, using a conversational AI assistant grounded in a custom knowledge base of technical manuals and maintenance logs.

Core Objectives:

  • Prescriptive Power: To move beyond prediction by using a RAG pipeline and a Large Language Model (LLM) to provide technicians with actionable, step-by-step guidance for diagnostics and repairs.
  • Conversational AI: To enable users to interact with the system using natural language. The "Troubleshooting Assistant" on each asset's detail page serves as the direct interface to the dynamic knowledge base, allowing users to ask complex questions and receive precise, context-aware answers.
  • Knowledge-Enhanced Decisions: To create a dynamic knowledge base from unstructured text (manuals, logs) using a vector database (ChromaDB), enhancing the AI's reasoning capabilities.
  • Adaptivity (Human-in-the-Loop): To build a system designed for continuous learning by capturing user feedback on AI-generated responses, creating a path for future model fine-tuning.
  • Explainable AI (XAI): To maintain trust by using SHAP to provide clear, human-understandable explanations for the underlying predictive models (RUL and Fault Classification).
  • End-to-End MLOps: To demonstrate a complete, production-ready workflow, from data ingestion and model training to a containerized (Docker) web application.

Technology Stack:

  • Backend: Python, Flask, Gunicorn
  • LLM & RAG: LangChain, Ollama, ChromaDB, Sentence-Transformers
  • Machine Learning & NLP: Scikit-learn, XGBoost, LightGBM, SHAP, NLTK
  • Data Handling: Pandas, NumPy
  • Frontend: HTML, Pico.css, Plotly.js, JavaScript
  • Containerization: Docker
